Laboratorio Módulo 11
Expresiones Regulares

Este laboratorio contiene dos ejercicios independientes para practicar expresiones regulares:

  • Ejercicio A: Validador de códigos IBAN españoles
  • Ejercicio B: Extractor de enlaces de imágenes desde HTML

Cada ejercicio está en su propia carpeta con sus archivo independientes.

Ejercicio A - Validador IBAN

Objetivo: Validar y analizar códigos IBAN españoles usando expresiones regulares.

Funcionalidades:

  • Validar formato de IBAN (con espacios, guiones, o sin separadores)
  • Extraer información: país, banco, sucursal, cuenta, etc.
  • Identificar el nombre del banco
  • Mostrar todos los componentes del IBAN
🚀 Abrir Ejercicio A

Ejercicio B - Extractor de Imágenes

Objetivo: Extraer enlaces de imágenes desde contenido HTML usando expresiones regulares.

Funcionalidades:

  • Buscar todos los tags <img> en HTML
  • Extraer atributos: src, alt, title
  • Mostrar vista previa de las imágenes
  • Validar URLs de imagen
🚀 Abrir Ejercicio B
